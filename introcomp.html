A Linguagem de Programação C

Originalmente desenvolvida por Dennis Ritchie entre os anos de 1969 e 1972 na Bell Telephone Laboratories Bell Labs, a linguagem C foi idealizada com a finalidade de permitir a re-escrita do sistema operacional Unix utilizando uma linguagem de relativo alto nível sem necessariamente fazer uso de recurso da Assembly. Desde então ela tem se tornado uma das mais linguagens de programação mais importantes e amplamente utilizadas de todos os tempos.
